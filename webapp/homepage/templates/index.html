Operator - List View

This module provides a visual representation of operators in a list format. Each operator is displayed with an avatar and a brief message.

Module Structure

  1. Operator Card List
    • Operator Avatar Container

      Contains the avatar image and text representation of the operator.

    • Operator Message

      A brief message or description associated with the operator.

Styling and Fonts

The module utilizes custom fonts for enhanced typography:

  • Kantumruy Pro - Used for general text styling.
  • Judson - Used for headings and emphasis.

Assets

  • Operator Avatar Image: Located at /static/operator_circle.svg
  • CSS Stylesheet: Located at /static/css/abop.css

Usage

This module is intended to be used in applications where a list of operators needs to be displayed, allowing users to quickly identify and understand the roles of different operators.
