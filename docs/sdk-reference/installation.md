# Installation

```bash
pip install concrete[openai]
```

## Extras

openai is an extra package that includes the OpenAI API client.

Other extras include:

- requests
- auth
- google-auth

## Sidecar packages

- `concrete-db`: For saving messages
- `concrete-async`: For asynchronous workers
